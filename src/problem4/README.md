# Problem 4: Sum to N

Compute the sum of integers from 1 through `n`.

Example: `sum_to_n(5)` → `15`

## Run

```bash
npm install
npm start
```

## Implementations

| Export       | Approach        | Time | Space |
|--------------|-----------------|------|-------|
| `sum_to_n_a` | formula         | O(1) | O(1)  |
| `sum_to_n_b` | while loop      | O(n) | O(1)  |
| `sum_to_n_c` | recursive helper| O(n) | O(n)  |

See [NOTES.md](./NOTES.md) for brief analysis.
